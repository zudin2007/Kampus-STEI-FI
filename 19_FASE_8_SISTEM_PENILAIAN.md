## A. KRITERIA PENILAIAN - STANDARDIZE

### Standard Grading System (Wajib):

```
BOBOT PENILAIAN:
├─ Tugas/Quiz/Exercise: 30%
│  └─ Frequent assessment (weekly/biweekly)
├─ UTS (Midterm): 35%
│  └─ Week 8-9
└─ UAS (Final): 35%
   └─ Week 15-16

GRADE SCALE:
A  = 85-100  (4.0 GPA)
B+ = 80-84   (3.5 GPA)
B  = 75-79   (3.0 GPA)
C+ = 70-74   (2.5 GPA)
C  = 65-69   (2.0 GPA) ← MINIMUM COMPETENCY
D  = 60-64   (1.0 GPA)
E  = <60     (0.0 GPA) ← FAIL

MINIMUM REQUIREMENT:
- Grade C (2.0) untuk semua MK
- GPA ≥ 2.0 per semester
- Attendance ≥ 75%
```

### Alternative Grading (untuk MK tertentu, approval Kaprodi):

```
Project-Based:    Tugas 50%, Exam 50%
Portfolio-Based:  Portfolio 60%, Exam 40%
Research-Based:   Paper 70%, Presentation 30%

Requirements:
- HARUS align dengan CPMK
- HARUS ada rubric yang clear
- HARUS approved oleh Kaprodi sebelum semester
```

## B. ASSESSMENT DESIGN - DOSEN TRAINING

### Rapat/Workshop (20 JULI) - Topik:

1. **Assessment Alignment dengan CPMK**
   - Setiap assessment harus measure specific CPMK
   - Use rubric yang clear & objective
   - Avoid bias & subjectivity

2. **Formative vs Summative**
   - Formative: Regular feedback (quiz, discussion, exercise)
   - Summative: Final evaluation (midterm, final exam, project)
   - Balance keduanya untuk student learning

3. **Valid & Reliable Assessment**
   - Validity: Does it measure what we intend?
   - Reliability: Is it consistent?
   - Fairness: Same standard untuk semua students

4. **Rubric Design Workshop**
   - Template rubric untuk berbagai assessment type
   - Example: Essay, project, presentation, lab report
   - Hands-on: Dosen practice membuat rubric mereka

**Attendance:** Wajib untuk semua dosen  
**Certificate:** Diberikan untuk record

## F. SPECIAL CASES

### Academic Probation / Low GPA:

```
POLICY:
- GPA < 2.0 → Academic Probation
- Action: Dosen wali → interventional meeting
- Support: Academic advising, tutoring, counseling
- Monitoring: Check progress setiap 2 minggu
- Review: Improve atau more action?
```

### Make-up / Remedial Options:

```
MAKE-UP EXAM:
- Jika sakit/emergency: Request dengan surat keterangan
- Timeline: Dalam 1 minggu setelah exam regular
- Approval: Kaprodi + dosen
- Sama soal & rubric dengan exam regular

REMEDIAL:
- Untuk grade < C (failed)
- Option: Retake MK atau alternative remedial (jika ada)
- Cost: Mahasiswa bayar
- Max 1x per MK
```
