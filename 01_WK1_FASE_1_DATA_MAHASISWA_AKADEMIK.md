# FASE 1: DATA MAHASISWA & REGISTRASI AKADEMIK
## WK1 TA 2026/2027 - Master Documentation

**Status:** READY TO EXECUTE
**Timeline:** 1-14 Juli 2026 (Minggu 1-2)
**Owner:** WK1 + Admin Akademik + Bagian Registrasi
**Priority:** CRITICAL - data mahasiswa adalah dasar seluruh proses akademik

---

## OVERVIEW

Fase 1 berfokus pada:
1. **Verifikasi data mahasiswa baru** - biodata, berkas registrasi ulang, input SIAKAD
2. **Verifikasi data mahasiswa lama** - status reguler/cuti/DO, kesiapan skripsi mahasiswa tingkat akhir
3. **Data kelulusan & yudisium** - validasi nilai, transkrip, persiapan wisuda
4. **Sinkronisasi PDDIKTI** - seluruh data mahasiswa wajib sinkron sebelum akhir Agustus

---

## DELIVERABLES

### A1: Verifikasi Data Mahasiswa Baru
**Deadline:** Minggu 1 Juli
**PIC:** Admin Akademik + Bagian Registrasi
**Status:** ☐

**Checklist:**
- ☐ Jumlah dan biodata mahasiswa yang diterima dicek terhadap data Admisi/PMB
- ☐ Kelengkapan berkas pendaftaran ulang diverifikasi (KTP, Ijazah, SKHU, foto)
- ☐ Data diinput/diupdate di SIAKAD dan sistem akademik lainnya
- ☐ Mahasiswa yang belum melengkapi dokumen teridentifikasi (buat daftar)
- ☐ Pengingat kelengkapan dokumen dikirim via Admin (email/WA)

**If data PMB belum lengkap:**
- 🚨 Koordinasi langsung dengan WK3 (pemilik proses PMB) — jangan menunggu
- Tetapkan batas akhir pelengkapan dokumen sebelum hari pertama kuliah

---

### A2: Verifikasi Data Mahasiswa Lama
**Deadline:** Minggu 1 Juli
**PIC:** Admin Akademik
**Status:** ☐

**Proses:**
1. Cek status akademik seluruh mahasiswa tahun sebelumnya (reguler/cuti/DO)
2. Identifikasi mahasiswa berisiko DO atau akan mengajukan cuti
3. Update seluruh status di SIAKAD
4. Verifikasi kesiapan mahasiswa tingkat akhir untuk skripsi/TA (koordinasi Kaprodi)

**Output:**
- Rekap status mahasiswa per angkatan per prodi
- Daftar mahasiswa berisiko (input untuk sistem early warning — lihat Panduan Kritis D5)

---

### A3: Data Kelulusan & Yudisium
**Deadline:** Sesuai jadwal yudisium
**PIC:** Admin Akademik + Ketua
**Status:** ☐

**Checklist:**
- ☐ Daftar mahasiswa peserta yudisium terverifikasi
- ☐ Nilai dan transkrip akademik tervalidasi
- ☐ Dokumentasi wisuda dipersiapkan

---

### A4: Sinkronisasi PDDIKTI
**Deadline:** Sebelum akhir Agustus (mulai dicicil Juli)
**PIC:** Admin Akademik + IT
**Status:** ☐

**Checklist:**
- ☐ Data mahasiswa baru siap diinput ke PDDIKTI
- ☐ Perubahan status mahasiswa lama dilaporkan
- ☐ Tidak ada selisih data antara SIAKAD dan PDDIKTI

---

## SUCCESS METRICS

✅ 100% mahasiswa baru terdaftar di SIAKAD dengan berkas lengkap
✅ Seluruh status mahasiswa lama terverifikasi & terupdate
✅ Daftar mahasiswa berisiko tersedia untuk early warning
✅ Rencana sinkronisasi PDDIKTI terjadwal
