## I. TUJUAN

3. ✅ Siap untuk mengikuti perkuliahan tanpa hambatan administratif

## V. PROSES VERIFIKASI STEP BY STEP

### STEP 1: PERSIAPAN DATA (Hari Ke-1)

**Aktivitas:**
1. Terima daftar mahasiswa dari Bagian Admisi
   ```
   File: "Daftar_Mahasiswa_Baru_2026.xlsx"
   - NIU
   - Nama lengkap
   - Prodi
   - Jalur masuk
   - Email & No. HP
   ```

2. Cross-check dengan data yang sudah di SIAKNG
   - [ ] Bandingkan daftar Admisi dengan SIAKNG
   - [ ] Identifikasi duplikasi atau error
   - [ ] Catat discrepancy di file "Issue_Log.xlsx"

3. Siapkan form verifikasi per mahasiswa
   - [ ] Print/copy form verifikasi (01_FORM_VERIFIKASI_MAHASISWA_BARU.md)
   - [ ] Label dengan NIU dan nama
   - [ ] Siapkan folder penempatan dokumen

**Output:** 
- ✅ Daftar mahasiswa terverifikasi
- ✅ Form verifikasi untuk setiap mahasiswa
- ✅ Issue log untuk discrepancy

**Owner:** Admin Akademik  
**Durasi:** 1 hari kerja

---

### STEP 2: PENGUMPULAN DOKUMEN (Hari 1-3)

**Aktivitas:**
1. Hubungi mahasiswa untuk mengumpulkan dokumen
   - [ ] Kirim email daftar dokumen yang diperlukan
   - [ ] Beri deadline pengumpulan (3-5 hari)
   - [ ] Sediakan multiple channel: offline + online upload

2. Terima dokumen dari mahasiswa
   - **Offline:** Datang ke kantor Admin Akademik
   - **Online:** Upload via Google Form atau portal akademik
   - **Email:** Kirim scan ke admin.akademik@stei.ac.id

3. Verifikasi kelengkapan dokumen yang masuk
   - [ ] Cek setiap dokumen per checklist
   - [ ] Identifikasi dokumen yang tidak jelas
   - [ ] Catat status di form verifikasi

**Template Notifikasi:**
```
Gunakan: 03_TEMPLATE_NOTIFIKASI_MAHASISWA.md
- Section A: Status Belum Lengkap (jika ada dokumen kurang)
- Section B: WA Reminder Awal (setelah 3 hari)
```

**Output:**
- ✅ Dokumen terkumpul dari ___ mahasiswa
- ✅ Form verifikasi diisi untuk setiap mahasiswa
- ✅ Daftar mahasiswa dengan dokumen incomplete

**Owner:** Admin Akademik + Bagian Admisi  
**Durasi:** 3-5 hari

---

### STEP 3: VERIFIKASI DOKUMEN (Hari 4-7)

**Aktivitas:**
1. Lakukan verifikasi mendetail setiap dokumen
   ```
   Checklist per dokumen:
   ☐ KTP: 
     - Asli atau fotokopi? → [Catat status]
     - Data sesuai dengan biodata?
     - Masih berlaku (tdk expired)?
   
   ☐ Ijazah SMA:
     - Nomor ijazah terbaca?
     - Sesuai dengan tahun lulus?
     - Tanda tangan & stempel jelas?
   
   ☐ SKHU:
     - Data nilai ada semua?
     - Stempel resmi ada?
     - Tanda tangan kepala sekolah ada?
   
   [DST untuk dokumen lainnya]
   ```

2. Entry data ke SIAKNG
   - [ ] Buka SIAKNG → Modul Akademik → Mahasiswa
   - [ ] Cari mahasiswa by NIU
   - [ ] Update field yang masih kosong/error:
     - Nama (harus sesuai ijazah)
     - Alamat
     - Email
     - No. HP
     - Data ortu
     - Asal sekolah
   - [ ] Simpan (Save) setiap update

3. Identifikasi dan catat masalah
   - [ ] Jika data tidak sesuai dokumen → Flag untuk koreksi
   - [ ] Jika dokumen tidak jelas → Catat untuk diminta ulang
   - [ ] Jika ada kekurangan → Catat untuk follow-up

**Checkllist Verifikasi:**
```
Gunakan: 01_FORM_VERIFIKASI_MAHASISWA_BARU.md
- Fill lengkap setiap section
- Catat status final (LENGKAP/PENDING/MASALAH)
```

**Output:**
- ✅ Semua dokumen terverifikasi
- ✅ Data di SIAKNG sudah update
- ✅ Issue log diupdate dengan status

**Owner:** Admin Akademik  
**Durasi:** 3-5 hari

**Tips Efisiensi:**
- Buat shift verifikasi jika mahasiswa banyak
- Minimal 2 orang untuk cross-check
- Gunakan checklist untuk ensure konsistensi

---

### STEP 4: FOLLOW-UP & REMINDER (Hari 3-7)

**Untuk mahasiswa yang BELUM LENGKAP:**

1. Kirim reminder 1 (Hari ke-3)
   - Medium: Email + WA
   - Isi: Dokumen apa yang masih kurang, deadline
   - Template: 03_TEMPLATE_NOTIFIKASI_MAHASISWA.md (Format 1)
   - [ ] Catat waktu pengiriman

2. Follow-up personal (Hari ke-5)
   - [ ] Hubungi via telpon jika perlu
   - [ ] Dengarkan alasan keterlambatan
   - [ ] Bantu solusi (jika dokumen belum bisa dikumpulkan)

3. Final reminder (Hari ke-6)
   - Medium: WA (urgent tone)
   - Isi: Ini pengingat terakhir, jangan terlewat
   - Template: 03_TEMPLATE_NOTIFIKASI_MAHASISWA.md (Format 2)
   - [ ] Clear deadline: hari dan jam

**Output:**
- ✅ Semua mahasiswa sudah dihubungi min. 2x
- ✅ Log komunikasi tercatat
- ✅ Mahasiswa tahu deadline jelas

**Owner:** Admin Akademik  
**Durasi:** Ongoing selama fase verifikasi

---

### STEP 5: KOREKSI DATA & RESOLUSI MASALAH (Hari 5-10)

**Untuk Error di SIAKNG:**

1. Identifikasi error detail
   ```
   Tipe error umum:
   - Nama tidak sesuai → koreksi sesuai ijazah
   - Email typo → update dengan yang benar
   - No. HP salah → tanyakan no. yang benar
   - Alamat tidak lengkap → lengkapi dari KTP
   - Data ortu kosong → entry dari form yang diberikan
   ```

2. Hubungi mahasiswa untuk koreksi data
   - [ ] Kirim email: "Mohon verifikasi data Anda di sistem"
   - [ ] Attach form data untuk dikonfirmasi
   - [ ] Minta mereka update online atau datang ke kantor

3. Lakukan koreksi di SIAKNG setelah konfirmasi
   - [ ] Buka SIAKNG → Edit data mahasiswa
   - [ ] Update field yang salah
   - [ ] Catat siapa yang mengupdate dan jam berapa
   - [ ] Minta approve dari Kaprodi (jika major change)
   - [ ] Simpan & archive screenshot sebelum-sesudah

4. Verifikasi ulang hasil koreksi
   - [ ] Print/screenshot data setelah koreksi
   - [ ] Cross-check dengan dokumen asli
   - [ ] Pastikan sudah benar sebelum mark as complete

**Output:**
- ✅ Semua error teridentifikasi & dicatat
- ✅ Data sudah dikoreksi di SIAKNG
- ✅ Verifikasi ulang sudah dilakukan

**Owner:** Admin Akademik + IT Support  
**Durasi:** 3-5 hari

---

### STEP 6: FINALISASI & APPROVAL (Hari 8-10)

**Aktivitas:**

1. Compile tracking report
   - [ ] Update master tracking spreadsheet
   - [ ] Hitung persentase kelengkapan
   - [ ] Identifikasi mahasiswa yang 100% selesai
   - [ ] Generate laporan untuk Dekan

2. Kirim laporan status ke WK1
   - Template: 04_TEMPLATE_LAPORAN_VERIFIKASI_UNTUK_DEKAN.md
   - Frequency: 2x seminggu (Rabu & Jumat)
   - [ ] Catat ACK dari WK1

3. Approval dari Kaprodi
   - [ ] Susun daftar mahasiswa yang sudah verified lengkap
   - [ ] Bawa ke Kaprodi untuk final approval
   - [ ] Catat tanda tangan approval di file

4. Penyiapan untuk tahap berikutnya
   - [ ] List mahasiswa yang siap kuliah
   - [ ] Catat mahasiswa dengan status special (butuh follow-up)
   - [ ] Siapkan untuk koordinasi dengan dosen wali

**Output:**
- ✅ Laporan status final tersedia
- ✅ Approval dari Kaprodi terdokumentasi
- ✅ Daftar mahasiswa siap kuliah terbuat

**Owner:** Admin Akademik + WK1  
**Durasi:** 1-2 hari

---

### STEP 7: ARSIP & DOKUMENTASI (Hari 10-15)

**Aktivitas:**

1. Arsip dokumen fisik
   - [ ] Buat folder per mahasiswa
   - [ ] Masukkan semua form verifikasi asli
   - [ ] Masukkan fotokopi dokumen penting (KTP, Ijazah, dll)
   - [ ] Labeling folder dengan NIU & Nama
   - [ ] Tata rapi di lemari arsip
   - [ ] Buat inventory file

2. Arsip dokumen digital
   - [ ] Scan semua dokumen yang penting (jika belum ada)
   - [ ] Organize by folder: [PRODI]/[TAHUN]/[MAHASISWA]
   - [ ] Upload ke storage cloud/server dengan backup
   - [ ] Encrypt/protect akses (password jika perlu)
   - [ ] Buat backup copy

3. Dokumentasi proses
   - [ ] Archive semua form verifikasi (asli atau scan)
   - [ ] Archive master tracking spreadsheet
   - [ ] Archive laporan progress & final report
   - [ ] Catat lessons learned untuk improvement tahun depan

**Output:**
- ✅ Semua dokumen tersusun rapi & mudah dicari
- ✅ Backup tersedia di multiple location
- ✅ Dokumentasi proses lengkap untuk audit

**Owner:** Admin Akademik  
**Durasi:** 2-3 hari

## IX. TROUBLESHOOTING GUIDE

| Problem | Penyebab | Solusi |
|---------|---------|--------|
| Mahasiswa tidak bisa akses SIAKNG | Password lupa / Belum aktif | Hubungi IT untuk reset password |
| Data di SIAKNG berbeda dokumen | Entry error saat daftar | Hubungi mahasiswa untuk konfirmasi, koreksi setelah verifikasi |
| Dokumen tidak jelas/rusak | Scan jelek atau fotokopi buruk | Minta mahasiswa upload ulang atau datang ke kantor |
| Mahasiswa tidak bisa dihubungi | Nomor telpon/email salah | Hubungi melalui Admisi atau orang tua |
| Dokumen masih dalam pengiriman | Mahasiswa baru lulus/belum ambil ijazah | Terima dokumen pengganti dulu (surat keterangan), dokumen asli bisa nanti |
| Ada duplikasi data di SIAKNG | Registrasi ganda atau error sistem | Koordinasi dengan IT untuk merge data |

## X. TIPS & BEST PRACTICE

✅ **DO:**
- Mulai verifikasi dari hari pertama registrasi dibuka
- Dokumentasi setiap keputusan & masalah yang ditemukan
- Komunikasi proaktif ke mahasiswa (jangan menunggu komplain)
- Backup data berkala (hindari data loss)
- Cross-check dengan dokumen asli untuk accuracy

❌ **DON'T:**
- Terima dokumen fotokopi saja tanpa ada yang memegang asli
- Update SIAKNG tanpa persetujuan mahasiswa atau Kaprodi (untuk data major)
- Biarkan error di SIAKNG tidak terkoreksi
- Lupa dokumentasi proses (untuk audit & improvement)
- Buru-buru approve sebelum 100% sesuai dengan dokumen asli
