# CRITICAL ITEM DEEP DIVE: B4 CAP/STEMPEL STANDARDISASI
## Accreditation Red Flag: Fithrah vs Fitrah Consistency

---

## THE PROBLEM

🚨 **Real Risk:** Inconsistent institution name spelling in official documents is a compliance/accreditation issue.

Example:
- Cap institusi says: "STEI Fithrah Insani"
- Some documents say: "STEI Fitrah Insani"
- Some say: "STEI FI"
- Some say: "Sekolah Tinggi Ekonomi Islam Fithrah Insani"

**Accreditor sees:** Inconsistency = lack of professional standards = RED FLAG

---

## WHAT IS THE CORRECT NAME?

**Step 1: Find authoritative source**
- [ ] SK Pendirian institusi (dari Kemenag) — this is THE source of truth
- [ ] Read: Official institution name as written in SK
- [ ] Document: Exact spelling (including diacritics if any)
- [ ] Example: "Sekolah Tinggi Ekonomi Islam Fithrah Insani" atau "...Fitrah..."?

**Step 2: Verify against other official documents**
- [ ] Statuta terbaru — what name is used?
- [ ] Sertifikat akreditasi — what name?
- [ ] Website — what name?
- [ ] Bank account — what name?
- [ ] Tax ID / Pajak — what name?

**If inconsistent:** The SK document is authoritative. Use that name.

---

## EXECUTION: B4 CAP/STEMPEL AUDIT

### PHASE 1: AUDIT (Start minggu 2, complete by 10 Juli)

1. **Inventory semua cap/stempel yang ada:**
   - Cap institusi besar (untuk surat resmi)
   - Stempel kantor WK2
   - Stempel kantor WK1, WK3, WK4
   - Stempel UPT, Lab
   - Any other stempel institusi

2. **For each cap/stempel, check:**
   - What name is written?
   - Is spelling correct? (Fithrah? Fitrah? Which is official?)
   - Is logo correct?
   - Is it in good condition (ink clear, tidak rusak)?

3. **Create audit list:**
   | No | Item | Current Name | Condition | Correct? | Action |
   |----|----|---|---|---|---|
   | 1 | Cap institusi | STEI Fithrah Insani | Good | ✓ | Keep |
   | 2 | Stempel WK2 | STEI Fitrah Insani | OK | ✗ | Replace |
   | 3 | Lab stempel | STEI FI | Worn | ✗ | Replace |

### PHASE 2: DECISION (by 12 Juli)

Based on audit + SK verification:
- Decide: What is THE official name?
- Which cap/stempel are correct? Which need replacing?

### PHASE 3: ORDER NEW (by 14 Juli)

If any need replacing:
1. Contact pabrik cap/stempel lokal (biasanya ada di kota)
2. Provide spec:
   - Exact name (copy dari SK)
   - Logo design (get from branding/marketing unit)
   - Size: [diameter] mm
   - Type: Embosser (for paper) atau Stamp (with ink)?
   - Quantity: 2-3 cap per unit (backup)
3. Timeline: Rush order (1 minggu)
4. Approve cost + process PO

### PHASE 4: IMPLEMENT (by 21 Juli)

1. When new cap arrived:
   - Verify: Name is correct? Logo OK? Quality good?
   
2. Distribute to units:
   - Send to WK2, WK1, WK3, WK4 kepala unit
   - Train: "Gunakan cap baru ini, jangan cap lama"
   - Collect: Cap lama (retire — jangan dipakai lagi!)

3. Document:
   - Foto sebelum/sesudah
   - Inventory update (which units have new stempel)
   - Training record (siapa sudah dikasih tau)

## WHY THIS IS CRITICAL FOR AKREDITASI

Accreditor checks:
- Are official documents properly stamped?
- Is the name on stamp consistent across documents?
- Inconsistency = institutional carelessness = potential compliance issues

By fixing this minggu 2, you PREVENT accreditor finding inconsistencies later.
