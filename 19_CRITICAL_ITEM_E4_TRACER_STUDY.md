## E2-E4 EXECUTION: STEP BY STEP

### STEP 1: SURVEY DESIGN (Before E2 launch)

**Key questions to ask alumni:**

1. **Employment Status**
   - Sudah bekerja? (yes/no)
   - Kalau belum, why? (still job hunting? further study? taking break?)
   - Durasi dari wisuda ke first job? (months)

2. **Job Details** (if working)
   - Posisi/jabatan?
   - Nama perusahaan?
   - Industry/sektor?
   - Salary range? (optional, sometimes sensitive)

3. **Skills Relevance** (critical for curriculum!)
   - Apakah program kami prepare anda untuk pekerjaan ini?
   - Skill apa yang paling useful dari program?
   - Skill apa yang kurang dari program? (gap identification!)
   - Soft skill (communication, teamwork, problem-solving) = relevant?

4. **Overall Satisfaction**
   - Rate program 1-10 (how satisfied overall?)
   - Would you recommend to others?
   - Any feedback for curriculum improvement?

5. **Interest in Engagement**
   - Apakah anda interested jadi mentor untuk mahasiswa aktif?
   - Apakah anda interested berbagi pengalaman/talk di kampus?

**Survey format:**
- Multiple choice (easier to compile) + open-ended (richer feedback)
- Online form (Google Form / Typeform / Qualtrics) untuk easy distribution & analysis
- Estimated time: 5-10 menit (so response rate higher)
- Language: Bahasa Indonesia (alumni lebih comfortable)

### STEP 2: DISTRIBUTION (E2)

**Timeline:** Mid-July, continue through end of July

**Methods:**
1. **Email blast:** Send survey link ke database alumni (E1)
2. **Reminder #1:** 1 minggu setelah send (to boost response)
3. **Reminder #2:** 1 minggu sebelum deadline (final push)
4. **Social media:** Post di alumni group/Facebook (if ada)
5. **Face-to-face:** If any alumni event, ask in-person

**Target response rate:** 30-40% (realistic for surveys)
- 100 alumni → hope untuk 30-40 responses
- More = better, tapi 30-40% sudah meaningful untuk analysis

### STEP 3: RESPONSE COLLECTION & MANAGEMENT

**Timeline:** July, ongoing

**Admin task:**
- Monitor responses daily (how many coming in?)
- Send reminder jika slow (encourage participation)
- QA: Check responses for completeness (missing answer = problematic)
- Data cleaning: Remove duplicate, invalid entries

**Incentive (optional):**
- Raffle: Random alumni who complete survey dapat voucher / merchandise
- Recognition: Thank you message, appreciation
- Benefit: Tell alumni "Your feedback shapes our curriculum improvement"

### STEP 4: DATA ANALYSIS & COMPILATION (Late July)

**Once responses collected:**

**Step 4a: Quantitative Analysis**

```
Employment Status:
- Total responden: [X]
- Employed: [Y] ([Y/X]%)
- Further study: [Z] ([Z/X]%)
- Still job hunting: [W] ([W/X]%)
- Not employed (other reasons): [V]

Time to employment (dari wisuda):
- < 3 months: [A]%
- 3-6 months: [B]%
- 6-12 months: [C]%
- >12 months: [D]%
- Average: [E] months

Salary range:
- <5 juta: [A]%
- 5-10 juta: [B]%
- 10-15 juta: [C]%
- >15 juta: [D]%

Industry distribution:
- Finance: [X]%
- Business: [Y]%
- Education: [Z]%
- Other: [W]%

Skills relevance (rate 1-10):
- Technical skills: [SCORE]
- Soft skills: [SCORE]
- Leadership: [SCORE]
- Problem-solving: [SCORE]
```

**Step 4b: Qualitative Analysis**

*Apa skill yang alumni suka dari program?*
- Frequently mentioned: [Skill 1, Skill 2, Skill 3]
- Quote: "[Alumni name]: [positive quote about what learned]"

*Skill gap: Apa yang kurang?*
- Frequently mentioned: [Gap 1, Gap 2, Gap 3]
- Quote: "[Alumni name]: [quote about what needed in job but not learned]"

*Overall satisfaction feedback:*
- Positive themes: [Theme 1, Theme 2]
- Constructive feedback: [Suggestion 1, Suggestion 2]

### STEP 5: REPORT TO WK1

**Format:** 3-5 page report

**Content:**
1. **Executive Summary:** Key findings (1 page)
   - Employment rate
   - Key skills strengths
   - Key skill gaps
   - Recommendation for curriculum

2. **Detailed Findings:** (2-3 pages)
   - Quantitative data (tables, charts)
   - Qualitative themes with quotes
   - Analysis per prodi if applicable (ES vs PS)

3. **Curriculum Input for WK1:** (½ page)
   - "Based on alumni feedback, suggest adding [Skill/Course] to curriculum"
   - "Strengthen [Topic] because alumni report high relevance in jobs"
   - "Consider reviewing [Course] because alumni feedback weak relevance"

4. **Appendix:**
   - Full survey questions
   - Response statistics (total responses, demographics)
   - Sample quotes

**Submit to:** WK1 + copy to Dekan
**Timing:** By end July (latest 31 July)
**Format:** PDF + Excel (data file for WK1 to use)

## E4 RED FLAGS TO AVOID

🚫 **Low response rate** (<20% responsiveness)  
→ FIX: Multiple reminders, incentivize, extend deadline

🚫 **Survey questions too vague** ("Was program good?")  
→ FIX: Specific questions on skill relevance, gap identification

🚫 **Data not analyzed** (just submit raw responses)  
→ FIX: Compile into report format, highlight patterns

🚫 **Report not actionable** (pretty stats but no curriculum implication)  
→ FIX: Include "based on this, suggest ..." recommendations

🚫 **Report to WK1 too late** (after curriculum cycle closed)  
→ FIX: Submit end July (latest), so WK1 can plan curriculum changes for next TA
