## WHY THIS MATTERS

Dosen mengajar **tanpa kontrak legal** = 
- ❌ PDDIKTI violation (laporan beban dosen tidak valid)
- ❌ Akreditasi red flag (audit akan find ini)
- ❌ Payroll issue (bagaimana hitung honor tanpa kontrak?)
- ❌ Dosen legal risk (dosen bisa claim belum punya perjanjian)

**HARD DEADLINE: 14 JULI** — Semester mulai 1 September, kontrak harus sudah signed 3 minggu sebelumnya.

## TYPES OF DOSEN CONTRACTS NEEDED

### 1. DOSEN BARU (tidak pernah ada kontrak)
**Process:**
1. WK1 confirm: dosen ini akan mengajar TA ganjil? (course + SKS?)
2. Admin SDM issue: Surat penerimaan (formal offer letter)
3. Dosen review + tanda tangan surat penerimaan
4. Admin SDM prepare: Kontrak standar
   - Nama dosen, posisi, SKS mengajar
   - Periode: [bulan start] - [bulan end]
   - Besaran honor (rate per SKS)
   - Ketentuan: Attendance, grading, peraturan
5. Dosen + Ketua + Yayasan: Tanda tangan kontrak
6. Scan + upload ke database dosen

**Timeline:** Start minggu 1, Target sign minggu 2 (by 14 Juli)

### 2. DOSEN LAMA (ada kontrak lama, akan perpanjang)
**Process:**
1. WK1 confirm: Dosen ini akan mengajar lagi TA ganjil?
2. Admin SDM evaluate: Kinerja tahun lalu satisfactory?
   - Mengajar OK?
   - Tugas2 dikerjakan?
   - Tidak ada complain dari mahasiswa?
3. If satisfactory: Approve perpanjangan
4. Opsi A: Kontrak perpanjangan baru (full document sign again)
   - Lebih formal, lebih secure
5. Opsi B: Amendment ke kontrak lama (update date + periode saja)
   - Lebih cepat, jika hanya date/periode yang berubah
6. Dosen + Ketua + Yayasan: Tanda tangan
7. Scan + archive

**Timeline:** Start minggu 1, Target signed minggu 2

### 3. DOSEN YANG TIDAK JADI MENGAJAR
**Process:**
1. WK1 confirm: Dosen ini di-drop dari jadwal?
2. Admin SDM: Issue formal letter (sorry, tidak perlu mengajar TA ini)
3. Dokumentasi untuk file dosen
