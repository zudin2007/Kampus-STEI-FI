# CRITICAL GUIDE: B4 CAP & STEMPEL STANDARDISASI

**Owner:** Wakil Ketua II (Admin) + Sekretaris  
**Deadline:** 25 Juli 2026  
**Risk Level:** 🟠 HIGH (legal/administrative risk)  

---

## WHY THIS IS CRITICAL

Cap & stempel (official seals) = legal authority of all institutional documents:
- SK dosen, SK mahasiswa, ijazah, transkrip → all MUST have cap/stempel to be valid
- If stempel digunakan sembarang → fraud risk (fake certificates circulating)
- If stempel hilang → no dokumen valid → students dapat ijazah palsu (institution liable)
- International credibility depends on stempel legitimacy (for AQIP accreditation)

**Issue:** Inconsistent stempel usage → some certificates not recognized by industry/government

---

## WHAT NEEDS TO HAPPEN

### Phase 1: AUDIT EXISTING STAMPS (10-15 Juli)

**Goal:** Tahu seberapa banyak cap & stempel yang ada, kondisinya apa

**Checklist:**
1. Inventory all caps/stamps in Sekretariat
   - Institutional cap (STEI)
   - Faculty cap (FSEI)
   - Program cap (ES, PS, etc)
   - Rector signature stamp (for SK)
   - Registrar stamp (for certificates)
   
2. For each one, assess:
   - Condition: New / Good / Worn / Damaged / Lost?
   - Usage frequency: Daily / Weekly / Monthly / Rarely?
   - Responsible person: Who keeps it?
   - Last maintenance: When last serviced?

3. Document in inventory spreadsheet:
```
Stamp Type | Condition | In Use? | Where Kept | Responsible Person | Notes
STEI Cap | Good | Yes | Cabinet A, Locked | Sekretaris | Used for SK
ES Cap | Worn | Yes | Cabinet B | Admin Staff 2 | Consider replace
RD Stamp | Excellent | Yes | Safe, Sekretaris | Sekretaris | Just serviced
```

**Output:** Stamp inventory + condition report

---

### Phase 2: STANDARDIZATION (16-20 Juli)

**Goal:** Ensure consistent design & quality

**Standards to establish:**
1. **Design:** All caps/stamps must have consistent university logo, font, size
2. **Quality:** All stamps must produce clear, legible imprints
3. **Access control:** Only authorized persons (Sekretaris + 1 designated admin) can use
4. **Documentation:** Every stamp use logged (what doc, date, who)

**Process:**
1. Identify which stamps are non-standard (outdated design, poor quality)
2. Create replacement list
3. Order new stamps from vendor (lead time ~1 week)
4. Test new stamps before put into use
5. Retire old stamps (destroy if damaged, archive if historical)

**New stamp standards:**
- Size: [Standard size per institutional brand guidelines]
- Font: [Institutional font]
- Material: [Metal (durable) or rubber (flexible)]
- Design: [Latest institutional logo]

---

### Phase 3: SECURITY & CONTROL (21-25 Juli)

**Goal:** Prevent unauthorized use & fraud

**Implementation:**
1. **Physical security:**
   - Stamps stored in LOCKED cabinet (only Sekretaris has key)
   - Cabinet location: Sekretariat (high-traffic, secure room)
   - Backup location: Rector office (emergency backup)

2. **Access control:**
   - Only Sekretaris authorized to use (primary)
   - Designated admin can use if Sekretaris unavailable (with Sekretaris approval)
   - NO student/outsider can touch stamp
   - Request form required for any stamp use (signature of requester + approver)

3. **Usage log:**
   ```
   DATE | TIME | DOCUMENT TYPE | STAMP USED | QUANTITY | AUTHORIZED BY | NOTES
   7/25 | 10:30 | SK Dosen | STEI Cap | 1 | Sekretaris | For Dr. Budi SK
   7/25 | 14:00 | Ijazah | Registrar | 5 | Sekretaris | Batch for 2023 grads
   ```

4. **Regular audits:**
   - Monthly: Count stamps, verify no damage, check log for anomalies
   - Quarterly: Verify cap/stamp still in secure location, test print quality
   - Annually: Replace worn items, refresh inventory

---

### Phase 4: TRAINING & DOCUMENTATION (22-25 Juli)

**Goal:** Ensure everyone knows the rules

**Training sessions:**
1. **Sekretaris:** Detailed training on security, usage, maintenance (1 hour)
2. **Designated backup admin:** Usage & security procedures (30 min)
3. **Document requesters (dosen, staff):** How to request & what to expect (email briefing)

**Documentation:**
1. **SOP (Standard Operating Procedure) for Cap/Stempel Usage:**
   ```
   TITLE: SOP Penggunaan Cap dan Stempel Resmi STEI
   
   1. TUJUAN: Memastikan penggunaan cap/stempel hanya untuk dokumen sah
   2. TANGGUNG JAWAB: Sekretaris (primary custody), Designated Admin (backup)
   3. PROSEDUR:
      a) Requester submit form (via Sekretaris inbox)
      b) Sekretaris verify document & legitimacy
      c) Sekretaris apply cap/stempel
      d) Log usage in spreadsheet
      e) Requester sign receipt
   4. LARANGAN:
      - Jangan pinjam stempel untuk dibawa keluar
      - Jangan izinkan orang lain gunakan tanpa approval
      - Jangan simpan di tempat tidak aman
   5. AUDIT: Monthly check by Sekretaris, quarterly review by WK2
   ```

2. **Inventory spreadsheet:** Master file (Google Sheets or Excel) with usage history

3. **Emergency procedures:** What to do if stamp lost/damaged/misused

## QUALITY ASSURANCE

### Testing New Stamps:
Before put into use, test each stamp:
1. Apply to test paper
2. Check: imprint clear? Design correct? Size consistent?
3. Document result: PASS or FAIL
4. Only use if PASS

### Regular maintenance:
- Monthly: Inspect for cracks, wear
- Quarterly: Deep clean, re-ink if needed
- Annually: Replace if worn significantly
