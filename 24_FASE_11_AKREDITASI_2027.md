## D. EVIDENCE DOCUMENTATION

### How to Document Evidence:

```
STRONG EVIDENCE:
✓ Data-driven (quantified)
✓ Well-organized & easy to navigate
✓ Clearly labeled & cross-referenced
✓ Includes interpretation (not just raw data)
✓ Shows continuous improvement

EXAMPLE - WEAK TO STRONG:

WEAK:
"Faculty teaches well"
→ No evidence, subjective

BETTER:
"Student evaluations average 4.2/5"
→ Some evidence, but need context

STRONG:
"Faculty teaching evaluations:
- 2024: 4.0/5 (n=156)
- 2025: 4.1/5 (n=162)
- 2026: 4.2/5 (n=158)
Trend: Continuous improvement
Action taken: [DETAIL]"
→ Data, trend, interpretation, action
```
