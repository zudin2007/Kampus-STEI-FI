# CRITICAL GUIDE: SEMESTER MONITORING & CLASSROOM VISITS

**Owner:** Kaprodi  

---

## PURPOSE

Mid-semester monitoring ensures quality teaching is happening.

**Without monitoring:**
- Dosen might not be teaching (just give handouts)
- Learning outcomes not achieved
- Students complain too late to fix

---

## MONITORING PROTOCOL

**3x per semester:**
- Week 3: First check-in (is class running smoothly?)
- Week 9: Mid-semester (are learning outcomes being achieved?)
- Week 15: Final check (grading plan ready?)

**For each classroom visit:**
1. Observe 15-20 min of class
2. Check: Is RPS being followed? Quality of teaching?
3. Speak with ~3 students: "How's the class? Learning well?"
4. Debrief with dosen: "Well done. Any support needed?"

---

## INTERVENTION TRIGGERS

🔴 If observed:
- Attendance low
- Learning activities weak
- Assessment doesn't match CPMK
- Student complaints

→ Action plan with dosen to improve
